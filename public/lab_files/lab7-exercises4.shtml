Exercise : array mystery

Suppose that the array [1, 3, 6, 10] was passed as a parameter to the mystery method below. Fill in the boxes with the array contents after the method call. 

public static void mystery(int[] data) {
    for(int index = data.length - 1; index > 0; index--) {
        data[index] = data[index] - data[index - 1];
    }
}
