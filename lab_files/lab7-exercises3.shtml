Exercise : array mystery

Suppose that the array and ints at right were passed as parameters to the mystery method below. Fill in the boxes with the array contents after the method call. 

public static void mystery(int[] a, int x, int y) {
    for (int i = 0; i < a.length; i++) {
        if (x < y) {
            y = y + 1;
            a[y] = 17;
        } else {
            a[x] = 42;
        }
    }
}
int[] a = {0, 0, 0, 0};
int x = 3;
int y = 1;
