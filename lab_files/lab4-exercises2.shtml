Exercise : if/else mystery

Consider the following Java code. Fill in the boxes with the output produced by each of the method calls. 

public static void mystery(int n) {
    if (n > 10) {
        n = n / 2;
    } else {
        n = n + 7;
    }
    if (n * 2 < 25) {
        n = n + 10;
    }
    System.out.println(n);
}
mystery(40); 
20
mystery(8); 
15
mystery(0); 
17
mystery(12); 
16
mystery(20); 
20
